To set up a local development environment:

1. Install all the required tools. This includes:
    a. Python 2.7.11 (https://www.python.org/downloads/release/python-2711/)
    b. Virtualenv 13.1.0 (pip install virtualenv)
    c. Git (http://git-scm.com/downloads)
2. Create a Virtualenv and populate it. In the root of the repo run:
    a. virtualenv venv
    b. venv\Scripts\activate.bat
    c. pip install --requirement=requirements.txt
    d. pip install --requirement=dev-requirements.txt
3. If needed, run 'python secret_key.py' from the root of the repo to generate a secret key and set the SECRET_KEY setting in web/newdjangosite/settings_base.py to be the output of the previous command
4. Set up the local database. From the /web directory run:
    a. python manage.py migrate
    b. python manage.py createsuperuser
5. Run the site using 'python manage.py runserver'
